# JS Calculator  
Calculadora hecha en JavaScript puro sin usar eval(), con operaciones básicas y orden de prioridad (*, /, +, -).

## 🚀 Funcionalidades:
- Encendido y apagado (ON / OFF)
- Operaciones con jerarquía
- Botón DEL y AC
- Estilos con CSS (no frameworks)

## 📁 Tecnologías usadas:
- HTML
- CSS / SCSS
- JavaScript Vanilla
